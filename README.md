## ğŸš€ Installation

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- A text editor (VS Code, Sublime Text, Atom, etc.)

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/Pliatsikas/bio-page.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd bio-page
   ```

3. **Open the project**
   - Simply open the `index.html` file in your web browser
   - OR use a local development server (optional):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (with http-server package)
     npx http-server
     ```

4. **View the page**
   - If opened directly: The page will load in your browser
   - If using a server: Navigate to `http://localhost:8000` in your browser

## ğŸ¨ Customization

To personalize the bio page with your own information:

1. Open `index.html` in your text editor
2. Update the content sections with your personal information:
   - Name and title
   - Bio description
   - Contact information
   - Links to your social media profiles
3. Modify `styles.css` (or your CSS file) to change:
   - Colors and fonts
   - Layout and spacing
   - Styling of individual elements
4. Save your changes and refresh the browser to see updates
